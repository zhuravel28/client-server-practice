# client-server-practice

Практика: клієнт-серверна архітектура

## Структура проекту:
Даний проєкт реалізує простий клієнт–серверний веб-додаток на основі Node.js та Express, у якому клієнт взаємодіє з сервером через HTTP-запити:

GET / → повернення HTML-сторінки

POST /data → передача JSON-даних клієнт → сервер

GET /status → отримання службової інформації про сервер
